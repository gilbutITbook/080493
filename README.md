# <A2A × MCP 멀티에이전트 오케스트레이션 실전> 소스 코드
  
## 실습 시 주의사항
실습 시 다음 내용을 참고하기 바랍니다.</br></br>
● 모든 실행 결과는 LLM 특성상 책과 다를 수 있습니다.</br>
● 커서가 때때로 도구(Tool)를 사용하지 않고 답변하는 경우가 있습니다. 이런 상황에서는 “해당 도구를 사용해 다시 답변해 달라”고 요청해 대화를 이어가면 됩니다. </br>
● 커서에서 답변이 영어로 출력될 경우, 질문에 “한글로 답변해 줘”를 덧붙여 다시 질문해 주세요. </br>
● CSV나 PDF 파일 참조를 위해 다음과 같이 절대 경로를 사용한 경우가 있습니다. 이 경로는 예제 파일을 저장해 둔 자신의 현재 파일 경로로 수정해 주세요.
(예시: C:\Users\JYSEO\A2A\) </br>
● 이 책은 에이전트 간 협업 구조를 이해하는 데 초점을 두고 있어, 흐름 설명에 집중하기 위해 벡터 기반 검색은 대부분의 예제에서 제외했습니다. 벡터 기반 RAG 구축에 대해서는 관련 전문 서적을 참고하시기 바랍니다.</br>
